# SoundBuddy

Native desktop audio companion for **Windows** — live visualizer overlay, music control flyout, now playing, volume mixer, and taskbar integration.

Built with **Tauri v2** (Rust) and **Angular 20** (TypeScript). The Rust workspace holds audio, media, volume, and preset logic; the WebView UI hosts flyout, overlay, settings, and mixer surfaces.
